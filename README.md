## Quick Open Selection

### Available commands

* `quickOpenSelection.quickOpenSelection` Quick Open with prefix from the active editor selection

### Keyboard Shortcuts

* `ctrl+p, ctrl+e` on Windiws and Linux
* `cmd+p` on macOS
